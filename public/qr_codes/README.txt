Інструкції по роботі з QR-кодами:

1. Для кожного сайту використовуйте відповідний файл:
   - сайт1.png - для першого сайту
   - сайт2.png - для другого сайту
   - сайт3.png - для третього сайту

2. Щоб замінити QR-код:
   - Просто завантажте новий файл з тим самим ім'ям
   - Або використовуйте кнопку "Upload New QR" в адмін-панелі

3. Вимоги до файлів:
   - Тільки формат PNG
   - Максимальний розмір: 5MB
   - Рекомендоване розширення: 300x300 пікселів

4. Безпека:
   - Регулярно оновлюйте QR-коди
   - Перевіряйте правильність завантажених кодів
   - Не діліться доступом до цієї директорії

При виникненні проблем зверніться до технічної підтримки.
